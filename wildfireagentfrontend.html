Setup Instructions

  1. Install required packages: pip install flask flask-cors pillow tensorflow
  2. Save the Python server code as wildfire_server.py
  3. Place your wildfire_detection_model_final.h5 file in the same folder
  4. Run the server: python wildfire_server.py
  5. The server will start on http://localhost:5000
